## 💻 Projeto
Este é um projeto web responsivo de uma cápsula do tempo para exibir memórias em uma linha do tempo.

## 🚀 Tecnologias
Foi desenvolvido durante o NLW da RocketSeat com as seguintes tecnologis:

- HTML
- CSS
- Git e Github
